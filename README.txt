QUIRKS: *
---------------------------------------------
Christen Eagle II has liveries zipped w/  description.ext file,
therefore to keep this mod light, it's livery unlocks have been
omitted.

F-16C also has many zipped liveries which have been omitted.

F-5E-3 has a livery folder with Cyrillic text, which breaks
OVGME's ZIP file extraction. (FiAF 11th FS Lapland Air Command)

The folder and livery are ignored in this mod, so that when
playing as anyone other than FINLAND in an F-5E, that livery
will NOT appear in the rearm/refuel livery selection dropdown.

If you want either of these things in your mod, use the larger
mod: Unlock Aircraft Liveries Full.
